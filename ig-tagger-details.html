Instagram Tagger 💬 - Automate people tagging in giveaways



Project information

Instagram tagger is a Python script that automatically logs into your Instagram account, navigates to the giveaway URL and starts tagging people in the comments section. You only have to provide it with your account credentials (username and password), the giveaway URL, the number of tags per comment and finally the list with the names of the followers you wish to tag. The script will automatically choose X random names from the followers list for each different comment, where X is the number of tags required by the giveaway.



Github repo: https://github.com/stavros-melidoniotis/ig_tagger
